Pretraining configs for the MIMIC-III dataset from HoTPP.
### Сохранение средних направлений градиента

В проекте предусмотрена возможность сохранять средние направления градиента в процессе обучения.

**Вариант 1. Через конфигурационный файл**

Для этого необходимо раскомментировать секцию `save_grad_params` в файле `default_hpo.yaml` и задать нужные параметры:

* `begin` — шаг, с которого начинается сохранение,
* `end` — шаг, на котором сохранение заканчивается,
* `name` — имя файла с сохранёнными градиентами.

Все параметры можно настраивать напрямую в этом блоке.

**Вариант 2. Через параметры командной строки**

Альтернативно параметры можно переопределить при запуске обучения через Hydra, например:

```
HYDRA_FULL_ERROR=1 python -m hotpp.train --config-dir configs --config-name ntp_cls_hpo_rnn +module.hpo_params.save_grad_params.end=100
```

Этот способ удобен, если не хочется изменять конфигурационные файлы напрямую или требуется быстрый эксперимент с параметрами.
